# IoT Weather Station - Group 11

This project is a comprehensive IoT Weather Station for monitoring indoor environmental parameters. The system utilises a Raspberry Pi Pico W for collecting data, processing it locally and for transmitting it wirelessly to a mobile application for visualisation and historical tracking.
